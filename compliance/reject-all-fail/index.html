Reject-All-Fail Test Site

Domain key: reject-all-fail

Purpose: A test site where one non-essential cookie is dropped only on the first user interaction (click / scroll / mousemove), rather than on page load. A passive scan that does not interact with the page does not observe this cookie, so it never enters the scanned cookie inventory — but any flow that clicks a button on the page will trigger it.

The stray cookie: _gcl_aw (Google Ads style, advertising / non-essential, matches the _gcl_* pattern). It is set with a long expiry and is not tied to any consent category on the page, so it persists regardless of the consent choice.
